## 1. 测试策略

### 1.1 测试分层

```
┌─────────────────────────────────────────────────┐
│  E2E 测试（Playwright）                          │
│  端到端用户流程验证                               │
├─────────────────────────────────────────────────┤
│  前端测试（Jest + RTL + Vitest）                 │
│  组件渲染 / 交互逻辑 / API 集成                  │
├─────────────────────────────────────────────────┤
│  后端接口测试（JUnit 5 + MockMvc）              │
```

## 2. 测试用例

## 3. 接口测试用例详情

### 3.1 接口：GET /api/v1/requests/{requestId}/overview

**请求示例**：
```bash
curl -X GET "http://localhost:9280/api/v1/requests/req_abc123/overview" \
  -H "Accept: application/json"
```

**成功响应**：
```json
{
  "code": 0,
  "data": {
    "requestId": "req_abc123",
    "topAgentName": "DataAnalysisAgent",
    "success": false,
    "totalTaskCount": 12,
    "failedTaskCount": 2,
    "totalDuration": 45230,
    "createTime": "2026-06-25T10:00:00",
    "agentInstances": [
      {
        "agentId": "agent_001",
        "agentName": "DataAnalysisAgent",
        "entrance": true,
        "taskCount": 8
      },
      {
        "agentId": "agent_002",
        "agentName": "SubSearchAgent",
        "entrance": false,
        "taskCount": 4
      }
    ]
  }
}
```

**失败场景**：requestId 不存在时返回空 data 而非 404，code 仍为 0。

### 3.2 接口：GET /api/v1/requests/{requestId}/log

**请求示例**：
```bash
curl "http://localhost:9280/api/v1/requests/req_abc123/log?username=alice&page=0&pageSize=5000"
```

**成功响应**：
```json
{
  "code": 0,
  "data": {
    "fileName": "req_abc123.log",
    "fileSize": 2048000,
    "startLine": 0,
    "endLine": 5000,
    "hasMore": true,
    "lines": [
      "2026-06-25 10:00:01.123 [main] INFO  traceId=req_abc123 - Agent started",
      "2026-06-25 10:00:01.456 [pool-1] INFO  traceId=req_abc123 - Executing task: analyze_data"
    ]
  }
}
```

## 5. 异常测试用例

| 用例ID | 场景 | 输入 | 预期行为 |
|--------|------|------|----------|
| ET-01 | 数据库连接超时 | 模拟 DB 不可用 | 重试 2 次后返回部分数据 + 友好错误提示 |
| ET-02 | 日志文件读取权限不足 | 无读权限 | 返回 403，提示权限不足 |
| ET-03 | requestId 注入攻击 | `requestId=../../etc/passwd` | 路径规范化解，不允许目录遍历 |
| ET-04 | 超大 page 参数 | `page=99999999` | 限制最大 pageSize，返回空列表 |
| ET-05 | 日志文件编码异常 | 非 UTF-8 编码文件 | 降级处理，尝试 GBK 编码 |
| ET-06 | LLM 调用记录缺失 | callId 存在但 body 为空 | 返回记录但 body 显示为空 |
| ET-07 | 跨月目录不存在 | 日志已归档清理 | 尝试 3 个月内所有月份，失败则提示不可用 |
