## 4. 风险清单

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|----------|
| llm-agent 日志表结构变更 | 中 | 高 | 接口层做好字段兼容，Schema 变更前沟通 |
| 超大日志文件内存溢出 | 中 | 高 | 分页加载 + 流式读取严格限制 |
| 源码对照跨仓库引用 | 中 | 中 | 提供手动指定脚本路径的能力 |
| 预发数据权限不满足 | 低 | 中 | 提前申请测试账号和权限 |
| PostgreSQL `$1` 占位符兼容 | 中 | 高 | 使用 NamedParameterJdbcTemplate 统一处理 |
| MongoDB 大文档查询 OOM | 中 | 中 | 限制 pageSize 最大 500，超大集合加索引 |
| AI API Key 未配置导致报错 | 低 | 低 | AI_ENABLED=false 默认关闭，异常不阻塞查询 |
| Spring Boot 4 第三方库兼容性 | 低 | 中 | 提前验证 Knife4j jakarta 版等关键依赖 |
